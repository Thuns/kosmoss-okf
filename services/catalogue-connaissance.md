# Le Catalogue de Connaissance

## En une phrase

Kosmoss construit et maintient pour votre entreprise une base de connaissance structurée en format OKF — le standard ouvert de Google — pour que les agents IA (ChatGPT, Perplexity, Gemini) vous citent correctement lorsqu'un client potentiel pose une question dans votre domaine.

## Le problème que ce service résout

Votre entreprise a 10, 20 ou 30 ans d'expertise dans vos têtes, dans vos documents internes et dans l'expérience de vos employés. Mais si quelqu'un demande aujourd'hui à ChatGPT de lui recommander un fournisseur dans votre secteur à Val-d'Or ou en Abitibi, l'IA répond avec les entreprises dont la connaissance est déjà structurée et accessible — pas nécessairement les meilleures, mais les mieux documentées pour les agents.

Ce service comble cet écart.

## Ce qui est livré

Un **bundle OKF** comprenant typiquement :

- **Identité organisationnelle** — qui vous êtes, votre mission, vos valeurs
- **Services et expertises** — description précise de chaque service, zone couverte, clientèle cible
- **FAQ opérationnelle** — les 20-40 questions que vos clients posent réellement
- **Processus documentés** — comment vous travaillez, de la demande initiale à la livraison
- **Tarification et modalités** — ce que les agents IA peuvent communiquer directement
- **Études de cas** — projets représentatifs (avec permission client)
- **Playbooks** (palier Autorité) — procédures opérationnelles pour vos agents IA internes

## Comment c'est livré

Le bundle est hébergé sur votre site web (dossier `/okf/`) et sur un dépôt GitHub public — deux points d'accès pour les agents IA. Il est également accompagné d'un fichier `llms.txt` à la racine de votre site.

## Résultat attendu

Dans les 3 à 6 mois suivant la mise en ligne, votre entreprise commence à apparaître dans les réponses des agents IA lorsqu'un utilisateur pose une question liée à votre secteur et votre région. Ce n'est pas du SEO — c'est de la **présence dans l'économie des agents**.
